# example-rest-api-fastify-mysql

## How to Install
1. npm install
2. create .env file
```
  SERVER_PORT=3000
  DB_HOST=localhost
  DB_USER=root
  DB_PASSWORD=root
  DB_NAME=depok_news
  DB_PORT=3306
```
3. node server.js
